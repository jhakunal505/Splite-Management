RoomieKhata Pro is a premium, glassmorphism-inspired web application designed by Head Developer Kunal Kumar Jha to simplify shared living expenses. It features a sophisticated "Smart Settlement" engine that automatically calculates who owes whom, eliminating manual math and roomie conflicts. The app offers a high-tech dark interface with a persistent transaction history log and an integrated calendar for tracking spending dates. Built with a mobile-first responsive architecture, it ensures seamless expense management across smartphones, tablets, and desktops. By utilizing local storage technology, it provides a secure and private experience, keeping all financial data directly on the user's device without needing a complex backend.# Splite-Management
